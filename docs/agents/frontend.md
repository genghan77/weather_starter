# Frontend Conventions

- Frontend code is React + Tailwind.
- Frontend state lives in `frontend/src/state/store.tsx`.
- Frontend entry point is `frontend/src/main.tsx`.
- When adding UI features, prefer new components under `frontend/src/components`.
- Keep styling in `frontend/src/index.css` or the component file.
- Key UI components currently include `Hero.tsx`, `Sidebar.tsx`, `Tiles.tsx`, `MapCard.tsx`, and `ThemeSelector.tsx`.
