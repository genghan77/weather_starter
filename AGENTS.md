# AGENTS.md

Weather Starter is a fullstack TypeScript weather app starter with an Express backend, a React/Vite frontend, and SQLite persistence.

## Essentials

- Package manager: `npm`
- Dev server: `npm run dev`
- Frontend build: `npm --prefix frontend run build`
- Frontend typecheck: `npx tsc -p frontend/tsconfig.json --noEmit`
- See the focused guidance in:
  - [Backend Conventions](docs/agents/backend.md)
  - [Frontend Conventions](docs/agents/frontend.md)
  - [Testing and DB Workflows](docs/agents/testing-and-db.md)
  - [Repo Operations](docs/agents/repo-operations.md)
